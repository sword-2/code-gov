This page is under construction. It reads a list of liaisons with an option to select which programming language formatted (sorted) the liaison's data ahead of time, as statically stored on GitHub. The programming language had to read json format and convert to a HTML fragment with a file extension *.l14.

More languages take more time, so little is completed initially.

The languages from companies or organizations was not integrated with the advertised [Web Assembly](https://webassembly.org/) effort. The Web Assembly was not understood well enough to do it. The web page's JavaScript only retrieves a static file prepared by a language. While JavaScript is capable of reading a JSON direct, it was not know if it would count as a memory safe language.

The *.l14 file extension represent an incomplete project. It came from Luke 14:28-30, but of that passage's context was not JavaScript, HTML, memory safe languages, or code liaisons.
